# ICU Admission Prediction – Big Data Analytics

This project predicts ICU admission risk using machine learning and big data techniques.

## Dataset
200,000+ healthcare records with 22 features.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- PySpark
- Matplotlib
- Seaborn

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Machine Learning Models
5. Clustering Analysis

## Models Used
- Logistic Regression
- Random Forest
- K-Means Clustering

## Project Structure

data/ → Dataset  
notebooks/ → Jupyter analysis notebook  
report/ → Project report  
requirements.txt → Python dependencies
